# [75. Sort Colors](https://leetcode.com/problems/sort-colors/description/)<Badge type="warning" text="Medium" />

Given an array <code>nums</code> with <code>n</code> objects colored red, white, or blue, sort them **in-place** so that objects of the same color are adjacent, with the colors in the order red, white, and blue.

We will use the integers <code>0</code>, <code>1</code>, and <code>2</code> to represent the color red, white, and blue, respectively.

You must solve this problem without using the library's sort function.

**Example 1:**

```
Input: nums = [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]
```

**Example 2:**

```
Input: nums = [2,1,0]
Output: [0,1,2]
```

**Constraints:**

- <code>n == nums.length</code>
- <code>1 <= n <= 300</code>
- <code>nums[i]</code> is either <code>0</code>, <code>1</code>, or <code>2</code>.

**Follow up:** Could you come up with a one-pass algorithm using only constant extra space?

## 💡 Approach 1: Dutch National Flag (One Pass, O(1) Space)

### Intuition

Use three pointers to partition the array into three sections: <code>[0s | 1s | 2s]</code> in a single pass.

- <code>low</code>: rightmost position for 0s (exclusive), initially -1
- <code>mid</code>: current index, traverses the array
- <code>high</code>: leftmost position for 2s (exclusive), initially n

### Algorithm

1. If <code>nums[mid] == 0</code>: swap with <code>nums[low+1]</code>, increment <code>low</code> and <code>mid</code>.
2. If <code>nums[mid] == 2</code>: swap with <code>nums[high-1]</code>, decrement <code>high</code> (do **not** increment <code>mid</code> — the swapped element may be 0 or 1).
3. If <code>nums[mid] == 1</code>: just increment <code>mid</code>.

### Key Points

- One pass, O(n) time, O(1) space.
- When swapping 2 to the end, the element brought from the right might be 0, 1, or 2 — so we must re-check <code>mid</code>.

## Code

::: code-group

```typescript [TypeScript]
function sortColors(nums: number[]): void {
    let low = -1, mid = 0, high = nums.length;
    while (mid < high) {
        if (nums[mid] === 0) {
            low++;
            [nums[low], nums[mid]] = [nums[mid], nums[low]];
            mid++;
        } else if (nums[mid] === 2) {
            high--;
            [nums[mid], nums[high]] = [nums[high], nums[mid]];
        } else {
            mid++;
        }
    }
}
```

```kotlin [Kotlin]
class Solution {
    fun sortColors(nums: IntArray): Unit {
        var low = -1
        var mid = 0
        var high = nums.size
        while (mid < high) {
            when (nums[mid]) {
                0 -> {
                    low++
                    nums[low] = nums[mid].also { nums[mid] = nums[low] }
                    mid++
                }
                2 -> {
                    high--
                    nums[mid] = nums[high].also { nums[high] = nums[mid] }
                }
                else -> mid++
            }
        }
    }
}
```

```java [Java]
class Solution {
    public void sortColors(int[] nums) {
        int low = -1, mid = 0, high = nums.length;
        while (mid < high) {
            if (nums[mid] == 0) {
                low++;
                swap(nums, low, mid);
                mid++;
            } else if (nums[mid] == 2) {
                high--;
                swap(nums, mid, high);
            } else {
                mid++;
            }
        }
    }
    void swap(int[] a, int i, int j) {
        int t = a[i]; a[i] = a[j]; a[j] = t;
    }
}
```

```swift [Swift]
class Solution {
    func sortColors(_ nums: inout [Int]) {
        var low = -1, mid = 0, high = nums.count
        while mid < high {
            switch nums[mid] {
            case 0:
                low += 1
                nums.swapAt(low, mid)
                mid += 1
            case 2:
                high -= 1
                nums.swapAt(mid, high)
            default:
                mid += 1
            }
        }
    }
}
```

## 💡 Approach 2: Counting Sort (Two Pass)

### Intuition

Count occurrences of 0, 1, 2; overwrite the array in order. Simple but requires two passes.

---

## Code (Counting Sort)

::: code-group

```typescript [TypeScript]
function sortColors(nums: number[]): void {
    const count = [0, 0, 0];
    for (const n of nums) count[n]++;
    let i = 0;
    for (let c = 0; c <= 2; c++)
        for (let j = 0; j < count[c]; j++) nums[i++] = c;
}
```

```kotlin [Kotlin]
class Solution {
    fun sortColors(nums: IntArray): Unit {
        val count = IntArray(3)
        for (n in nums) count[n]++
        var i = 0
        for (c in 0..2)
            repeat(count[c]) { nums[i++] = c }
    }
}
```

## Complexity

- **Dutch National Flag:** Time O(n), Space O(1) — one pass
- **Counting Sort:** Time O(n), Space O(1) — two passes
