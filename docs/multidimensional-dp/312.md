# [312. Burst Balloons](https://leetcode.com/problems/burst-balloons/description/) <Badge type="danger" text="Hard" />

You are given <code>n</code> balloons, indexed from <code>0</code> to <code>n - 1</code>. Each balloon is painted with a number on it represented by an array <code>nums</code>. You are asked to burst all the balloons.

If you burst the <code>i</code>-th balloon, you will get <code>nums[i - 1] * nums[i] * nums[i + 1]</code> coins. If <code>i - 1</code> or <code>i + 1</code> goes out of bounds of the array, treat it as if there is a balloon with a <code>1</code> there.

Return *the maximum coins you can collect by bursting the balloons wisely*.

**Example 1:**

```
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] --> [3,5,8] --> [3,8] --> [8] --> []
coins =  3*1*5    +   3*5*8  +  1*3*8  + 1*8*1 = 15 + 120 + 24 + 8 = 167
```

**Example 2:**

```
Input: nums = [1,5]
Output: 10
```

**Constraints:**

- <code>n == nums.length</code>
- <code>1 <= n <= 300</code>
- <code>0 <= nums[i] <= 100</code>

## 💡 Approach: Interval DP

### Idea

- “Which balloon to burst first” is hard because boundaries change. Fix **which balloon is burst last** in a range instead: if balloon <code>k</code> is the last one burst among those strictly between <code>i</code> and <code>j</code>, then everything in <code>(i, k)</code> and <code>(k, j)</code> is already gone, and bursting <code>k</code> yields <code>nums[i] * nums[k] * nums[j]</code> (virtual <code>1</code>s at the ends are handled by padding).
- Pad the array: <code>a = [1] + nums + [1]</code>, length <code>m = n + 2</code>.
- Define <code>dp[i][j]</code> = maximum coins from bursting **all** original balloons **strictly between** indices <code>i</code> and <code>j</code> (endpoints <code>i</code> and <code>j</code> are not burst; they act as fixed neighbors).
- For every interval <code>(i, j)</code> with <code>j - i >= 2</code>, try each <code>k</code> with <code>i &lt; k &lt; j</code> as the last balloon burst in that interval:
  ```
  dp[i][j] = max over k ( dp[i][k] + a[i] * a[k] * a[j] + dp[k][j] )
  ```
- Answer: <code>dp[0][m - 1]</code>.

```typescript [TypeScript]
class Solution {
    maxCoins(nums: number[]): number {
        const a = [1, ...nums, 1];
        const m = a.length;
        const dp: number[][] = Array(m)
            .fill(null)
            .map(() => Array(m).fill(0));

        for (let len = 2; len < m; len++) {
            for (let i = 0; i + len < m; i++) {
                const j = i + len;
                for (let k = i + 1; k < j; k++) {
                    dp[i][j] = Math.max(
                        dp[i][j],
                        dp[i][k] + a[i] * a[k] * a[j] + dp[k][j]
                    );
                }
            }
        }

        return dp[0][m - 1];
    }
}
```

```java [Java]
class Solution {
    public int maxCoins(int[] nums) {
        int n = nums.length;
        int[] a = new int[n + 2];
        a[0] = 1;
        a[n + 1] = 1;
        System.arraycopy(nums, 0, a, 1, n);
        int m = a.length;
        int[][] dp = new int[m][m];

        for (int len = 2; len < m; len++) {
            for (int i = 0; i + len < m; i++) {
                int j = i + len;
                for (int k = i + 1; k < j; k++) {
                    dp[i][j] = Math.max(
                        dp[i][j],
                        dp[i][k] + a[i] * a[k] * a[j] + dp[k][j]
                    );
                }
            }
        }

        return dp[0][m - 1];
    }
}
```

```python [Python]
class Solution:
    def maxCoins(self, nums: List[int]) -> int:
        a = [1] + nums + [1]
        m = len(a)
        dp = [[0] * m for _ in range(m)]

        for length in range(2, m):
            for i in range(m - length):
                j = i + length
                for k in range(i + 1, j):
                    dp[i][j] = max(
                        dp[i][j],
                        dp[i][k] + a[i] * a[k] * a[j] + dp[k][j],
                    )

        return dp[0][m - 1]
```

# ⏱️ Time & Space Complexity

| Complexity Type | Big-O | Explanation |
|-----------------|-------|---------------|
| **Time** | `O(n³)` | Intervals `O(n²)`, inner `k` loop `O(n)`. |
| **Space** | `O(n²)` | DP table of size `(n + 2)²`. |

Where `n` is `nums.length` (padded length is `n + 2`).
