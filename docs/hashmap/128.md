# [128. Longest Consecutive Sequence](https://leetcode.com/problems/longest-consecutive-sequence/description/) <Badge type="warning" text="Medium" />

Given an unsorted array of integers <code>nums</code>, return the length of the longest **consecutive elements sequence**.

You must write an algorithm that runs in <code>O(n)</code> time.

**Example 1:**

```
Input: nums = [100,4,200,1,3,2]
Output: 4
Explanation: The longest consecutive sequence is [1, 2, 3, 4]. Its length is 4.
```

**Example 2:**

```
Input: nums = [0,3,7,2,5,8,4,6,0,1]
Output: 9
Explanation: The longest consecutive sequence is [0, 1, 2, 3, 4, 5, 6, 7, 8].
```

**Constraints:**

- <code>0 <= nums.length <= 10<sup>5</sup></code>
- <code>-10<sup>9</sup> <= nums[i] <= 10<sup>9</sup></code>

## 💡 Approach: HashSet — Only Count from Sequence Start

### Intuition

Use a **HashSet** for O(1) lookup. To achieve O(n) time, only start counting from a number that is the **start of a consecutive sequence** — i.e., <code>num - 1</code> is not in the set. Then extend rightward (<code>num + 1</code>, <code>num + 2</code>, ...) until the sequence breaks.

### Algorithm

1. Put all elements into a set.
2. For each <code>num</code>, if <code>num - 1</code> is **not** in the set, then <code>num</code> is a sequence start.
3. Count consecutive elements: <code>num</code>, <code>num + 1</code>, <code>num + 2</code>, ... until the next value is missing.
4. Update the maximum length.

```typescript [TypeScript]
function longestConsecutive(nums: number[]): number {
    const set = new Set(nums);
    let maxLen = 0;

    for (const num of set) {
        if (!set.has(num - 1)) {
            let curr = num;
            let len = 0;
            while (set.has(curr)) {
                len++;
                curr++;
            }
            maxLen = Math.max(maxLen, len);
        }
    }

    return maxLen;
}
```

```kotlin [Kotlin]
class Solution {
    fun longestConsecutive(nums: IntArray): Int {
        val set = nums.toSet()
        var maxLen = 0

        for (num in set) {
            if (num - 1 !in set) {
                var curr = num
                var len = 0
                while (curr in set) {
                    len++
                    curr++
                }
                maxLen = maxOf(maxLen, len)
            }
        }

        return maxLen
    }
}
```

```java [Java]
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int num : nums) set.add(num);

        int maxLen = 0;
        for (int num : set) {
            if (!set.contains(num - 1)) {
                int curr = num;
                int len = 0;
                while (set.contains(curr)) {
                    len++;
                    curr++;
                }
                maxLen = Math.max(maxLen, len);
            }
        }

        return maxLen;
    }
}
```

```python [Python]
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        num_set = set(nums)
        max_len = 0

        for num in num_set:
            if num - 1 not in num_set:
                curr = num
                length = 0
                while curr in num_set:
                    length += 1
                    curr += 1
                max_len = max(max_len, length)

        return max_len
```

## Complexity

- **Time:** O(n) — each element is processed at most twice (once as start, once in a chain)
- **Space:** O(n) — HashSet
