# [413. Arithmetic Slices](https://leetcode.com/problems/arithmetic-slices/description/) <Badge type="warning" text="Medium" />

An integer array is called **arithmetic** if it consists of **at least three elements** and if the difference between any two consecutive elements is the same.

- For example, <code>[1,3,5,7,9]</code>, <code>[7,7,7,7]</code>, and <code>[3,-1,-5,-9]</code> are arithmetic sequences.

Given an integer array <code>nums</code>, return *the number of **arithmetic subarrays** of* <code>nums</code>.

**Example 1:**

```
Input: nums = [1,2,3,4]
Output: 3
Explanation: We have 3 arithmetic slices in nums: [1, 2, 3], [2, 3, 4] and [1, 2, 3, 4].
```

**Example 2:**

```
Input: nums = [1,2,3,8,9,10]
Output: 2
Explanation: We have 2 arithmetic slices in nums: [1, 2, 3] and [8, 9, 10].
```

**Constraints:**

- <code>1 <= nums.length <= 5000</code>
- <code>-1000 <= nums[i] <= 1000</code>

## 💡 Approach: 1D DP

### Intuition

- Define <code>dp[i]</code> = number of **arithmetic slices ending at index <code>i</code>** (the slice must include <code>nums[i]</code>).
- For <code>i >= 2</code>: if <code>nums[i] - nums[i-1] == nums[i-1] - nums[i-2]</code>, then we can extend every slice ending at <code>i-1</code> by one element, plus one new slice <code>[i-2, i-1, i]</code>. So <code>dp[i] = dp[i-1] + 1</code>.
- Otherwise <code>dp[i] = 0</code>.
- Answer = sum of all <code>dp[i]</code>.

### Transition

- <code>dp[i] = dp[i-1] + 1</code> if <code>nums[i] - nums[i-1] == nums[i-1] - nums[i-2]</code>, else <code>dp[i] = 0</code>.

## Code

::: code-group

```typescript [TypeScript]
function numberOfArithmeticSlices(nums: number[]): number {
    if (nums.length < 3) return 0;
    let dp = 0;
    let total = 0;
    for (let i = 2; i < nums.length; i++) {
        if (nums[i] - nums[i - 1] === nums[i - 1] - nums[i - 2]) {
            dp = dp + 1;
            total += dp;
        } else {
            dp = 0;
        }
    }
    return total;
}
```
