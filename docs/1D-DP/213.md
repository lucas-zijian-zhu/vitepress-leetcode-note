# [213. House Robber II](https://leetcode.com/problems/house-robber-ii/description/) <Badge type="warning" text="Medium" />

You are a robber planning to rob houses along a street. Each house has a certain amount of money. The houses are arranged in a **circle**, meaning the first house is the neighbor of the last one. You **cannot rob two adjacent houses** (it will trigger the alarm). Given an integer array <code>nums</code> where <code>nums[i]</code> is the amount at house <code>i</code>, return the **maximum** amount of money you can rob tonight.

**Example 1:**

```
Input: nums = [2,3,2]
Output: 3
Explanation: You cannot rob house 0 (index 0) and house 2 (index 2) because they are adjacent. Rob house 1 (index 1) to get 3.
```

**Example 2:**

```
Input: nums = [1,2,3,1]
Output: 4
Explanation: Rob house 0 (index 0) and house 2 (index 2), total 4.
```

**Example 3:**

```
Input: nums = [1,2,3]
Output: 3
```

**Constraints:**

- <code>1 <= nums.length <= 100</code>
- <code>0 <= nums[i] <= 1000</code>

## 💡 Approach: 1D DP (Split into Two Linear Cases)

### Intuition

Because the houses form a **circle**, we cannot rob both the first and last house. Split into two linear subproblems:

1. **Case A:** Rob houses <code>nums[0..n-2]</code> (exclude last house)
2. **Case B:** Rob houses <code>nums[1..n-1]</code> (exclude first house)

Return <code>max(caseA, caseB)</code>. Each case uses the same logic as [198. House Robber](/1D-DP/198).

### Edge case

- If <code>nums.length === 1</code>, return <code>nums[0]</code>.

```typescript [TypeScript]
function rob(nums: number[]): number {
    if (nums.length === 1) return nums[0];
    return Math.max(robRange(nums, 0, nums.length - 2), robRange(nums, 1, nums.length - 1));
}

function robRange(nums: number[], start: number, end: number): number {
    let prev = nums[start], curr = Math.max(nums[start], nums[start + 1] ?? nums[start]);
    for (let i = start + 2; i <= end; i++) {
        const next = Math.max(curr, prev + nums[i]);
        prev = curr;
        curr = next;
    }
    return curr;
}
```

```python [Python]
class Solution:
    def rob(self, nums: List[int]) -> int:
        if len(nums) == 1:
            return nums[0]
        return max(self._rob_range(nums, 0, len(nums) - 2), self._rob_range(nums, 1, len(nums) - 1))

    def _rob_range(self, nums: List[int], start: int, end: int) -> int:
        prev = nums[start]
        curr = max(nums[start], nums[start + 1]) if start < end else prev
        for i in range(start + 2, end + 1):
            prev, curr = curr, max(curr, prev + nums[i])
        return curr
```
