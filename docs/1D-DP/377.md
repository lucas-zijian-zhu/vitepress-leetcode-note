# [377. Combination Sum IV](https://leetcode.com/problems/combination-sum-iv/description/) <Badge type="warning" text="Medium" />

Given an array of **distinct** integers <code>nums</code> and a target integer <code>target</code>, return *the number of possible combinations that add up to* <code>target</code>.

The test cases are generated so that the answer can fit in a **32-bit** integer.

**Example 1:**

```
Input: nums = [1,2,3], target = 4
Output: 7
Explanation: The possible combination ways are:
(1, 1, 1, 1)
(1, 1, 2)
(1, 2, 1)
(1, 3)
(2, 1, 1)
(2, 2)
(3, 1)
Note that different sequences are counted as different combinations.
```

**Example 2:**

```
Input: nums = [9], target = 3
Output: 0
```

**Constraints:**

- <code>1 <= nums.length <= 200</code>
- <code>1 <= nums[i] <= 1000</code>
- All the elements of <code>nums</code> are **unique**.
- <code>1 <= target <= 1000</code>

## 💡 Approach: 1D DP (Count Ways)

### Intuition

- **Order matters**: <code>(1,1,2)</code> and <code>(2,1,1)</code> count as different combinations. So we iterate by **target** first, then for each amount try every number (like unbounded choices at each step).
- Define <code>dp[i]</code> = number of combinations that sum to <code>i</code>.
- <code>dp[0] = 1</code> (one way to get sum 0: use no numbers).
- For <code>i</code> from <code>1</code> to <code>target</code>: for each <code>num</code> in <code>nums</code>, if <code>i >= num</code>, then <code>dp[i] += dp[i - num]</code>.

### Transition

- <code>dp[i] += dp[i - num]</code> for each <code>num</code> in <code>nums</code> with <code>i >= num</code>.

## Code

::: code-group

```typescript [TypeScript]
function combinationSum4(nums: number[], target: number): number {
    const dp = new Array(target + 1).fill(0);
    dp[0] = 1;

    for (let i = 1; i <= target; i++) {
        for (const num of nums) {
            if (i >= num) {
                dp[i] += dp[i - num];
            }
        }
    }

    return dp[target];
}
```

class Solution {
    func combinationSum4(_ nums: [Int], _ target: Int) -> Int {
        var dp = Array(repeating: 0, count: target + 1)
        dp[0] = 1

        for i in 1...target {
            for num in nums {
                if i >= num {
                    dp[i] &+= dp[i - num]
                }
            }
        }

        return dp[target]
    }
}

```python [Python]
class Solution:
    def combinationSum4(self, nums: List[int], target: int) -> int:
        dp = [0] * (target + 1)
        dp[0] = 1

        for i in range(1, target + 1):
            for num in nums:
                if i >= num:
                    dp[i] += dp[i - num]

        return dp[target]
```

### ⏱️ Time and Space Complexity

| Complexity Type    | Value        | Explanation |
|--------------------|--------------|-------------|
| **Time Complexity** | `O(target * n)` | For each amount `1..target`, we consider every number in `nums`. |
| **Space Complexity**| `O(target)` | 1D `dp` array of size `target + 1`. |
