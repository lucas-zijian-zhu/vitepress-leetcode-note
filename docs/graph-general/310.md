# [310. Minimum Height Trees](https://leetcode.com/problems/minimum-height-trees/description/)<Badge type="warning" text="Medium" />

A tree is an undirected graph in which any two vertices are connected by exactly one path. Given a tree of <code>n</code> nodes labelled from <code>0</code> to <code>n - 1</code>, and an array of <code>n - 1</code> <code>edges</code> where <code>edges[i] = [a<sub>i</sub>, b<sub>i</sub>]</code> indicates an undirected edge between nodes <code>a<sub>i</sub></code> and <code>b<sub>i</sub></code>, you can choose any node as the root. The **height** of a rooted tree is the number of edges on the longest downward path between the root and a leaf.

Return a list of all **minimum height tree (MHT)** root labels. You can return the answer in **any order**.

**Example 1:**

```
Input: n = 4, edges = [[1,0],[1,2],[1,3]]
Output: [1]
Explanation: When the root is node 1, the tree height is 1; 1 is the only MHT root.
```

**Example 2:**

```
Input: n = 6, edges = [[3,0],[3,1],[3,2],[3,4],[5,4]]
Output: [3,4]
```

**Constraints:**

- The input is guaranteed to be a tree with no repeated edges.
- <code>1 <= n <= 2 * 10<sup>4</sup></code>
- <code>edges.length == n - 1</code>
- <code>0 <= a<sub>i</sub>, b<sub>i</sub> < n</code>, <code>a<sub>i</sub> != b<sub>i</sub></code>

## 🚀 Approach: Peel Leaves (BFS / Topological)

### Intuition

MHT roots are the **centers** of the tree: choosing them minimizes the longest path from root to a leaf. For a tree:

- There are at most **2** MHT roots (when the “center” is the midpoint of the longest path, i.e. an edge).
- We can find the center by **peeling layers from the outside in**: repeatedly remove all **leaves** (degree 1) until 1 or 2 nodes remain. Those are the MHT roots.

This is similar to topological sort on an undirected tree: process by “in-degree” (here, degree), and shrink from leaves inward.

### Algorithm

1. Build the graph (adjacency list) and compute each node’s **degree**.
2. Enqueue all **leaves** (degree === 1).
3. While there are more than 2 nodes left:
   - Process the current layer of leaves: for each leaf, remove it and decrease its neighbors’ degrees.
   - If a neighbor’s degree becomes 1, it is a leaf in the next layer — add it to the queue.
4. The remaining 1 or 2 nodes in the queue (or last processed) are the MHT roots.

### Key Points

- Handle <code>n === 1</code>: single node is the only root, return <code>[0]</code>.
- We peel until we have at most 2 nodes; the final layer (or the last node) gives the answer.

```typescript [TypeScript]
function findMinHeightTrees(n: number, edges: number[][]): number[] {
    if (n === 1) return [0];

    const graph: number[][] = Array.from({ length: n }, () => []);
    const degree = new Array(n).fill(0);

    for (const [a, b] of edges) {
        graph[a].push(b);
        graph[b].push(a);
        degree[a]++;
        degree[b]++;
    }

    const queue: number[] = [];
    for (let i = 0; i < n; i++) {
        if (degree[i] === 1) queue.push(i);
    }

    let remaining = n;
    while (remaining > 2) {
        const layerSize = queue.length;
        remaining -= layerSize;
        for (let i = 0; i < layerSize; i++) {
            const u = queue.shift()!;
            for (const v of graph[u]) {
                degree[v]--;
                if (degree[v] === 1) queue.push(v);
            }
        }
    }

    return queue;
}
```

```kotlin [Kotlin]
class Solution {
    fun findMinHeightTrees(n: Int, edges: Array<IntArray>): List<Int> {
        if (n == 1) return listOf(0)

        val graph = Array(n) { mutableListOf<Int>() }
        val degree = IntArray(n)

        for ((a, b) in edges) {
            graph[a].add(b)
            graph[b].add(a)
            degree[a]++
            degree[b]++
        }

        val queue = ArrayDeque<Int>()
        for (i in 0 until n) {
            if (degree[i] == 1) queue.offer(i)
        }

        var remaining = n
        while (remaining > 2) {
            val layerSize = queue.size
            remaining -= layerSize
            repeat(layerSize) {
                val u = queue.poll()
                for (v in graph[u]) {
                    degree[v]--
                    if (degree[v] == 1) queue.offer(v)
                }
            }
        }

        return queue.toList()
    }
}
```

```java [Java]
class Solution {
    public List<Integer> findMinHeightTrees(int n, int[][] edges) {
        if (n == 1) return List.of(0);

        List<List<Integer>> graph = new ArrayList<>();
        int[] degree = new int[n];
        for (int i = 0; i < n; i++) graph.add(new ArrayList<>());

        for (int[] e : edges) {
            int a = e[0], b = e[1];
            graph.get(a).add(b);
            graph.get(b).add(a);
            degree[a]++;
            degree[b]++;
        }

        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            if (degree[i] == 1) queue.offer(i);
        }

        int remaining = n;
        while (remaining > 2) {
            int layerSize = queue.size();
            remaining -= layerSize;
            for (int i = 0; i < layerSize; i++) {
                int u = queue.poll();
                for (int v : graph.get(u)) {
                    degree[v]--;
                    if (degree[v] == 1) queue.offer(v);
                }
            }
        }

        return new ArrayList<>(queue);
    }
}
```

```swift [Swift]
        if n == 1 { return [0] }

        var graph = Array(repeating: [Int](), count: n)
        var degree = Array(repeating: 0, count: n)

            remaining -= layerSize

            }
```

```python [Python]
class Solution:
    def findMinHeightTrees(self, n: int, edges: List[List[int]]) -> List[int]:
        if n == 1:
            return [0]

        graph = [[] for _ in range(n)]
        degree = [0] * n
        for a, b in edges:
            graph[a].append(b)
            graph[b].append(a)
            degree[a] += 1
            degree[b] += 1

        queue = deque(i for i in range(n) if degree[i] == 1)
        remaining = n
        while remaining > 2:
            layer_size = len(queue)
            remaining -= layer_size
            for _ in range(layer_size):
                u = queue.popleft()
                for v in graph[u]:
                    degree[v] -= 1
                    if degree[v] == 1:
                        queue.append(v)

        return list(queue)
```

- **Time complexity:** O(n) — each node and edge is processed a constant number of times.
