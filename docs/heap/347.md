# [347. Top K Frequent Elements](https://leetcode.com/problems/top-k-frequent-elements/description/)<Badge type="warning" text="Medium" />

Given an integer array <code>nums</code> and an integer <code>k</code>, return the <code>k</code> most frequent elements. You may return the answer in **any order**.

**Example 1:**

```
Input: nums = [1,1,1,2,2,3], k = 2
Output: [1,2]
```

**Example 2:**

```
Input: nums = [1], k = 1
Output: [1]
```

**Constraints:**

- <code>1 <= nums.length <= 10^5</code>
- <code>-10^4 <= nums[i] <= 10^4</code>
- <code>k</code> is in the range <code>[1, number of unique elements]</code>.
- The answer is **guaranteed to be unique**.

**Follow up:** Your algorithm's time complexity must be better than O(n log n).

## 💡 Approach 1: Min Heap (O(n log k))

### Intuition

1. Use a **hash map** to count frequency of each element.
2. Maintain a **min heap of size k** ordered by frequency; the root has the smallest frequency among the top k.
3. When heap size exceeds k, pop the smallest; after processing, the heap contains the k most frequent elements.

### Algorithm

1. Count frequencies: <code>map[num] = freq</code>.
2. For each <code>(num, freq)</code>, push <code>(freq, num)</code> into a min heap.
3. If heap size > k, pop the minimum (smallest freq).
4. Extract all elements from the heap into the result.

### Key Points

- Min heap compares by frequency; we keep the k largest frequencies.
- Time O(n log k), space O(n).

## Code

::: code-group

```typescript [TypeScript]
function topKFrequent(nums: number[], k: number): number[] {
    const freq = new Map<number, number>();
    for (const n of nums) freq.set(n, (freq.get(n) ?? 0) + 1);

    const minHeap = new MinPriorityQueue<[number, number]>({ priority: (x) => x[0] });
    for (const [num, count] of freq) {
        minHeap.enqueue([count, num]);
        if (minHeap.size() > k) minHeap.dequeue();
    }
    const result: number[] = [];
    while (!minHeap.isEmpty()) result.push(minHeap.dequeue().element[1]);
    return result;
}
```

```kotlin [Kotlin]
import java.util.PriorityQueue

class Solution {
    fun topKFrequent(nums: IntArray, k: Int): IntArray {
        val freq = nums.groupingBy { it }.eachCount()

        val minHeap = PriorityQueue<Pair<Int, Int>>(compareBy { it.first })
        for ((num, count) in freq) {
            minHeap.offer(count to num)
            if (minHeap.size > k) minHeap.poll()
        }
        return minHeap.map { it.second }.toIntArray()
    }
}
```

```java [Java]
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        Map<Integer, Integer> freq = new HashMap<>();
        for (int n : nums) freq.merge(n, 1, Integer::sum);

        PriorityQueue<int[]> minHeap = new PriorityQueue<>((a, b) -> a[0] - b[0]);
        for (var e : freq.entrySet()) {
            minHeap.offer(new int[]{e.getValue(), e.getKey()});
            if (minHeap.size() > k) minHeap.poll();
        }
        int[] result = new int[k];
        for (int i = 0; i < k; i++) result[i] = minHeap.poll()[1];
        return result;
    }
}
```

```swift [Swift]
class Solution {
    func topKFrequent(_ nums: [Int], _ k: Int) -> [Int] {
        var freq: [Int: Int] = [:]
        for n in nums { freq[n, default: 0] += 1 }

        var heap: [(Int, Int)] = []
        for (num, count) in freq {
            heap.append((count, num))
            heap.sort { $0.0 < $1.0 }
            if heap.count > k { heap.removeFirst() }
        }
        return heap.map { $0.1 }
    }
}
```

```python [Python]
import heapq

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        freq = {}
        for n in nums:
            freq[n] = freq.get(n, 0) + 1

        min_heap = []
        for num, count in freq.items():
            heapq.heappush(min_heap, (count, num))
            if len(min_heap) > k:
                heapq.heappop(min_heap)

        return [num for _, num in min_heap]
```

## 💡 Approach 2: Bucket Sort (O(n))

### Intuition

Create buckets where <code>bucket[i]</code> holds all numbers that appear exactly <code>i</code> times. Traverse from the highest frequency bucket downward and collect k elements.

### Algorithm

1. Count frequencies.
2. Create <code>bucket[freq] = list of nums</code>; max freq ≤ n.
3. From freq = n down to 1, append bucket elements to result until we have k.

## Code (Bucket Sort)

::: code-group

```typescript [TypeScript]
function topKFrequent(nums: number[], k: number): number[] {
    const freq = new Map<number, number>();
    for (const n of nums) freq.set(n, (freq.get(n) ?? 0) + 1);

    const buckets: number[][] = Array(nums.length + 1).fill(null).map(() => []);
    for (const [num, count] of freq) buckets[count].push(num);

    const result: number[] = [];
    for (let i = buckets.length - 1; i >= 0 && result.length < k; i--) {
        for (const num of buckets[i]) {
            result.push(num);
            if (result.length === k) return result;
        }
    }
    return result;
}
```

```kotlin [Kotlin]
        }

    }
```

```java [Java]
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        Map<Integer, Integer> freq = new HashMap<>();
        for (int n : nums) freq.merge(n, 1, Integer::sum);

        List<Integer>[] buckets = new List[nums.length + 1];
        for (int i = 0; i <= nums.length; i++) buckets[i] = new ArrayList<>();
        for (var e : freq.entrySet()) buckets[e.getValue()].add(e.getKey());

        int[] result = new int[k];
        int idx = 0;
        for (int i = nums.length; i >= 0 && idx < k; i--) {
            for (int num : buckets[i]) {
                result[idx++] = num;
                if (idx == k) return result;
            }
        }
        return result;
    }
}
```
