# [451. Sort Characters By Frequency](https://leetcode.com/problems/sort-characters-by-frequency/description/)<Badge type="warning" text="Medium" />

Given a string <code>s</code>, sort it in **decreasing order** based on the **frequency** of the characters. The frequency of a character is the number of times it appears in the string. Return the sorted string. If there are multiple answers, return any of them.

**Example 1:**

```
Input: s = "tree"
Output: "eert"
Explanation: 'e' appears twice while 'r' and 't' both appear once.
So 'e' must appear before both 'r' and 't'. Therefore "eetr" is also a valid answer.
```

**Example 2:**

```
Input: s = "cccaaa"
Output: "aaaccc"
Explanation: Both 'c' and 'a' appear three times, so "aaaccc" is also valid.
```

**Example 3:**

```
Input: s = "Aabb"
Output: "bbAa"
Explanation: "bbaA" is also valid, but "Aabb" is incorrect.
```

**Constraints:**

- <code>1 <= s.length <= 5 * 10^5</code>
- <code>s</code> consists of uppercase and lowercase English letters and digits.

## 💡 Approach 1: Max Heap (O(n log n))

### Intuition

1. Use a **hash map** to count frequency of each character.
2. Use a **max heap** ordered by frequency; the root has the highest frequency.
3. Pop from heap one by one and append each character repeated by its frequency.

### Algorithm

1. Count frequencies: <code>map[char] = freq</code>.
2. For each <code>(char, freq)</code>, push <code>(freq, char)</code> into a max heap.
3. Pop all elements, append <code>char</code> repeated <code>freq</code> times.

### Key Points

- Max heap compares by frequency; we process highest freq first.
- Time O(n log n), space O(n).

## Code

::: code-group

```typescript [TypeScript]
function frequencySort(s: string): string {
    const freq = new Map<string, number>();
    for (const c of s) freq.set(c, (freq.get(c) ?? 0) + 1);

    const maxHeap = new MaxPriorityQueue<[number, string]>({ priority: (x) => x[0] });
    for (const [char, count] of freq) maxHeap.enqueue([count, char]);

    const result: string[] = [];
    while (!maxHeap.isEmpty()) {
        const [count, char] = maxHeap.dequeue().element;
        result.push(char.repeat(count));
    }
    return result.join("");
}
```

```kotlin [Kotlin]
import java.util.PriorityQueue

class Solution {
    fun frequencySort(s: String): String {
        val freq = s.groupingBy { it }.eachCount()

        val maxHeap = PriorityQueue<Pair<Int, Char>>(compareByDescending { it.first })
        for ((char, count) in freq) maxHeap.offer(count to char)

        val sb = StringBuilder()
        while (maxHeap.isNotEmpty()) {
            val (count, char) = maxHeap.poll()
            repeat(count) { sb.append(char) }
        }
        return sb.toString()
    }
}
```

```java [Java]
class Solution {
    public String frequencySort(String s) {
        Map<Character, Integer> freq = new HashMap<>();
        for (char c : s.toCharArray()) freq.merge(c, 1, Integer::sum);

        PriorityQueue<int[]> maxHeap = new PriorityQueue<>((a, b) -> b[0] - a[0]);
        for (var e : freq.entrySet())
            maxHeap.offer(new int[]{e.getValue(), e.getKey()});

        StringBuilder sb = new StringBuilder();
        while (!maxHeap.isEmpty()) {
            int[] p = maxHeap.poll();
            sb.append(String.valueOf((char) p[1]).repeat(p[0]));
        }
        return sb.toString();
    }
}
```

```swift [Swift]
class Solution {
    func frequencySort(_ s: String) -> String {
        var freq: [Character: Int] = [:]
        for c in s { freq[c, default: 0] += 1 }

        var heap: [(Int, Character)] = []
        for (char, count) in freq {
            heap.append((count, char))
        }
        heap.sort { $0.0 > $1.0 }

        for (count, char) in heap {
            result += String(repeating: String(char), count: count)
        }
        return result
    }
```

```python [Python]
import heapq

class Solution:
    def frequencySort(self, s: str) -> str:
        freq = {}
        for c in s:
            freq[c] = freq.get(c, 0) + 1

        max_heap = [(-count, char) for char, count in freq.items()]
        heapq.heapify(max_heap)

        result = []
        while max_heap:
            neg_count, char = heapq.heappop(max_heap)
            result.append(char * (-neg_count))
        return "".join(result)
```

## 💡 Approach 2: Bucket Sort (O(n))

### Intuition

Create buckets where <code>bucket[i]</code> holds all characters that appear exactly <code>i</code> times. Traverse from the highest frequency bucket downward and append each character repeated by its frequency.

### Algorithm

1. Count frequencies.
2. Create <code>bucket[freq] = list of chars</code>; max freq ≤ n.
3. From freq = n down to 1, for each char in bucket, append char repeated freq times.

## Code (Bucket Sort)

::: code-group

```typescript [TypeScript]
function frequencySort(s: string): string {
    const freq = new Map<string, number>();
    for (const c of s) freq.set(c, (freq.get(c) ?? 0) + 1);

    const buckets: string[][] = Array(s.length + 1).fill(null).map(() => []);
    for (const [char, count] of freq) buckets[count].push(char);

    const result: string[] = [];
    for (let i = buckets.length - 1; i >= 0; i--) {
        for (const char of buckets[i]) result.push(char.repeat(i));
    }
    return result.join("");
}
```

```python [Python]
class Solution:
    def frequencySort(self, s: str) -> str:
        freq = {}
        for c in s:
            freq[c] = freq.get(c, 0) + 1

        buckets = [[] for _ in range(len(s) + 1)]
        for char, count in freq.items():
            buckets[count].append(char)

        result = []
        for i in range(len(buckets) - 1, -1, -1):
            for char in buckets[i]:
                result.append(char * i)
        return "".join(result)
```
